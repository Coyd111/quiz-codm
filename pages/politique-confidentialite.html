2.2 Données collectées automatiquement

Lors de votre utilisation du site, nous collectons automatiquement :

  • Données de navigation : pages visitées, durée de session, type de navigateur
  • Données techniques : adresse IP (anonymisée), résolution d'écran, système d'exploitation
  • Données de performance : temps de réponse, erreurs techniques

Important : Nous ne collectons JAMAIS :

  • Données sensibles (origine, religion, opinions politiques)
  • Données financières ou bancaires
  • Mots de passe ou données de connexion
  • Informations personnelles non nécessaires

  1. Fonctionnement du quiz :

    • Limitation quotidienne du quiz
    • Calcul des statistiques personnelles
  2. Tirage au sort mensuel :

  3. Système de parrainage :

3.2 Utilisation secondaire

  • Amélioration du service : analyse des performances, correction de bugs
  • Communication : informations importantes, résultats des tirages
  • Conformité légale : respect des obligations légales

4. Base légale du traitement

Nos traitements de données reposent sur :

Bases légales RGPD

  • Consentement (Art. 6.1.a) : Pour la participation au tirage au sort
  • Intérêt légitime (Art. 6.1.f) : Pour le fonctionnement du quiz et l'amélioration du service
  • Obligation légale (Art. 6.1.c) : Pour la conservation des données de tirage (transparence)

6. Sécurité des données

6.1 Mesures techniques

  • Chiffrement : HTTPS obligatoire pour toutes les communications
  • Stockage local : Données sensibles stockées uniquement sur votre appareil
  • Anonymisation : Suppression des identifiants des données statistiques
  • Accès limité : Seul l'administrateur peut accéder aux données

6.2 Mesures organisationnelles

  • Formation à la protection des données
  • Procédures de réponse aux incidents
  • Audits réguliers de sécurité
  • Documentation complète des traitements

8. Vos droits RGPD

Vos droits fondamentaux

8.1 Droit d'accès (Art. 15)

Vous pouvez demander une copie de toutes les données personnelles que nous détenons sur vous.

8.2 Droit de rectification (Art. 16)

Vous pouvez demander la correction de données inexactes ou incomplètes.

8.3 Droit à l'effacement (Art. 17)

Vous pouvez demander la suppression de vos données dans certaines circonstances.

8.4 Droit à la limitation (Art. 18)

Vous pouvez demander la limitation du traitement de vos données.

8.5 Droit à la portabilité (Art. 20)

Vous pouvez demander vos données dans un format structuré et lisible.

8.6 Droit d'opposition (Art. 21)

Vous pouvez vous opposer au traitement de vos données à tout moment.

8.7 Droit de retrait du consentement

Vous pouvez retirer votre consentement à tout moment sans affecter la légalité des traitements antérieurs.

16. Glossaire RGPD

Terme Définition
Données personnelles Toute information permettant d'identifier directement ou indirectement une personne physique
Responsable de traitement Personne qui détermine les finalités et les moyens du traitement des données
Consentement Accord libre, spécifique, éclairé et univoque de la personne concernée
Intérêt légitime Base légale permettant le traitement si l'intérêt est réel et ne porte pas atteinte aux droits des personnes
Anonymisation Processus rendant impossible l'identification d'une personne à partir des données
